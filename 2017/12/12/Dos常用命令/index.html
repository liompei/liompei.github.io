Dos常用命令

发表于 2017-12-12 | 分类于 基础知识 | | 阅读次数:

cls(clear screen) – 清屏

dir(directory) –列出当前文件夹下所有文件夹

  • dir/ah –仅显示隐藏文件

  • dir/as –仅显示系统文件

  • dir/ash –显示隐藏文件和系统文件

  • dir/? –显示所有可选项

cd aaaa –进入当前文件夹下aaaa目录中

cd.. –返回上一级

cd\ –返回到根目录

D: –进入D磁盘

help –查看帮助
