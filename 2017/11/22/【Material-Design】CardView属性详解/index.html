【Material Design】CardView属性详解

发表于 2017-11-22 | | 阅读次数:

CardView 卡片式布局,继承自FrameLayout,用户可以自定义圆角弧度和Z轴高度,阴影大小等

展示:一个简单的cardView

caedView属性详解:

  • app:cardBackgroundColor     背景颜色
  • app:cardCornerRadius       圆角弧度
  • app:cardElevation     阴影大小
  • app:cardMaxElevation     阴影的最大高度
  • app:cardPreventCornerOverlap     为内容添加内边距防止与CardView角点重叠(为兼容5.0之前版本)
  • app:cardUseCompatPadding     填充内容设置内边距(为5.0之后版本添加相同的填充值)
  • app:contentPadding     设置内边距
  • app:contentPaddingBottom
  • app:contentPaddingLeft
  • app:contentPaddingRight
  • app:contentPaddingTop
