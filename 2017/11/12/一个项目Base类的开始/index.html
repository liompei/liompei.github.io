项目Base类搭建

发表于 2017-11-12 | 分类于 Android | | 阅读次数:

前言

一个项目最基础的当然就是Application,BaseActivity,BaseFragment的搭建了,下面是我的写法,当做参考未尝不可,如果有更优的解决方案请多指教

Application

App.class

其中,activityList将每次打开的activity添加到list中,方便进行统一处理(如退出),当然在BaseActivity中进行配置后才会生效,下面看看BaseActivity

BaseActivity

在Activity onCreate时调用: App.getInstance().addActivity(this);
onDestroy时移除Activity: App.getInstance().deleteActivity(this);

  • getLayoutId()  传入Activity的布局文件
  • initViews(Bundle savedInstanceState)  初始化view控件,如findViewById
  • initData()  初始化数据
  • onEvent()  进行事件
  • toast()  用于toast输出
  • getToolbar(CharSequence title, boolean showHomeAsUp) 创建toolBar并返回一个toolbar对象

传入的值分别为:

toolbar的title,是否显示返回按钮,返回按钮由getSupportActionBar().setDisplayHomeAsUpEnabled(showHomeAsUp);设定,并在onOptionsItemSelected中添加了点击事件

Toolbar

使用Toolbar请先删除ActionBar,在styles.xml的AppTheme中配置

getToolbar需要在layout中创建一个包含toolbar的布局文件,例如
abc_toolbar.xml

  • background:  toolbar的背景色
  • titleTextAppearance:  toolbar字体颜色,大小,(加粗,斜体等)
  • theme:  toolbar上menu的样式
